Quilt Playground: TypeScript that writes TypeScript that writes HTML

A self-specializing live dashboard — three staged passes, all in the browser. Stage 1 (makeRenderer) runs once: it unrolls the metrics from the schema & opts panel into a flat, branch-free Stage 2 (start()) — the generated program shown below-left — whose own loop (interval baked in) repaints the HTML. The ↓ reduce operator runs that stage; then Stage 3 is the HTML the loop sets, with no further expansion. Editing the schema/opts panel (or the source) regenerates the TypeScript; the codegened loop stays cheap. The arrow glyphs are hard to type, so use the buttons — or the VS Code chords: ⌘/Ctrl+1 then a direction for a single glyph, ⌘/Ctrl+2 then two for a diagonal. ⌘/Ctrl+↵ expands & runs.
